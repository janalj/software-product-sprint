My Projects

Water Level API Request

Frontend: React, HTML, CSS, JS
Backend: Nodejs, Express, Vite
Description: A responsive dynamic website that sends AJAX requests based on month and year to the public data API from CDEC to obtain the water levels at the seven biggest California reservoirs. Upon data returned, the website generate the chart and presents to the users.

Favorite Pet Videos

Frontend: HTML, CSS, JS
Backend: Nodejs, Express, SQLite, Tiktok API
Description: A multi-player game for picking the winner pet video. Users upload their favorite Tiktok pet videos. The database stores the entered data. The application loads the videos for users to compare and like. Finally, it generates the winner video.
